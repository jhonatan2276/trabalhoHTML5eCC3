Input

Esse componente é um input personalizado que permite alterações de cor e tamanho dos mesmos.

<div class="input-group"> <input class="input-default" placeholder="Nome"/> </div>

Para utilizar as variações de cores acrescente as tags "input-black", "input-red", "input-yellow" ou "input-green" após a declaração do "input-default".

Dessa forma, a cor da tag adicionada será adicionada ao componente:

<div class="input-group"> <input class="input-default input-black" placeholder="Telefone"/> <input class="input-default input-red" placeholder="Endereço"/> <input class="input-default input-yellow" placeholder="Número"/> <input class="input-default input-green" placeholder="Complemento"/> </div>

Também é possível adicionar um input com um tamanho maior.

Para utilizá-lo basta acrescentar a tag "input-medium" ao invés do "input-default".

Observação: Para variações de cores o tipo médio funciona da mesma forma que o tipo default!

<div class="input-group"> <input class="input-medium" placeholder="Telefone" /> <input class="input-medium input-black" placeholder="Endereço"/> <input class="input-medium input-green" placeholder="Complemento"/> </div>

• Funciona para todos os tipos de input, não é ligado especificamente com types "text" ou qualquer outro. Use sem moderação!
